### 1. Data Pipeline
Documentation of the funnel from the original 31,739 reviews down to the 119 high-insight categorized reviews


| Stage | Description | Count (N) | % of Original |
| :--- | :--- | :---: | :---: |
| **Original** | The original dataset taken from Kaggle. | 31,739 | 100% |
| **Reviews** | Dataset filtered only for reviews. | 31,214 | 98.35% |
| **Filtered** | Dataset after filtering for only reviews which contained the words "anxious" or "anxiety". | 680 | 2.14% |
| **Sampled** | Ratings proportionate random sample taken from the Filtered dataset. | 200 | 0.63% |
| **Categorised** | Dataset only containing reviews with High-insight from the random sample. | 119 | 0.37% |
